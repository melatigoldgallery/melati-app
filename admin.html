Bantuan Sistem Antrian
Cara Menggunakan Sistem Antrian
  • Nomor Antrian Berikutnya - Klik tombol ini untuk mengambil nomor antrian berikutnya.
  • Panggil Nomor Antrian - Klik tombol ini untuk memanggil nomor antrian saat ini.
  • Sudah Dilayani - Klik tombol ini setelah pelanggan selesai dilayani.
  • Nomor Antrian Tertunda - Klik tombol ini jika pelanggan tidak hadir dan perlu ditunda.
  • Panggil Antrian Berurutan - Klik tombol ini untuk memanggil antrian tertunda secara berurutan.
  • Reset Nomor Antrian - Klik tombol ini untuk mereset nomor antrian ke awal.
Informasi Kendaraan

Anda dapat mencatat informasi kendaraan pelanggan untuk memudahkan identifikasi:

  • Pilih jenis kendaraan (Motor/Mobil)
  • Masukkan nomor plat kendaraan
  • Masukkan warna kendaraan
  • Klik "Simpan Informasi Kendaraan" untuk menyimpan data
Pengumuman

Sistem dilengkapi dengan dua jenis pengumuman:

  • Informasi Tunggu - Untuk memberitahu pelanggan agar bersabar menunggu.
  • Pengingat Antrian - Untuk mengingatkan pelanggan yang nomornya telah dipanggil.
